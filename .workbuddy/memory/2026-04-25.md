### 拖拽选中功能实现细节（2026-04-25 晚-补）

**核心机制**：
- `_blockKeys: Map<Block, GlobalKey>` — 为每个 Block 存储 GlobalKey，用于定位 RenderBox
- `_blockRects: Map<Block, Rect>` — 测量并缓存每个 Block 的全局边界
- `_updateBlockRects()` — 遍历 `_blockKeys` 获取 RenderBox，测量 `localToGlobal(Offset.zero)` 位置
- `_findBlockAtPosition(Offset)` — 根据全局坐标查找对应的 Block

**手势处理流程**：
1. `onLongPressStart` → `_updateBlockRects()` 刷新位置 → `operationControl.startLongPressSelect()`
2. `onLongPressMoveUpdate` → `_findBlockAtPosition()` 查找 Block → `operationControl.updateLongPressSelect()`
3. `onLongPressEnd` → `operationControl.endLongPressSelect()`
4. `onLongPressCancel` → `operationControl.cancelLongPressSelect()`

**OperationControl 长按逻辑**（loaders.dart）：
- `startLongPressSelect` → 初始化状态，清空旧选中，设置起点块
- `updateLongPressSelect` → 调用 `getBlocksByIndexRange()` 计算起点→当前块之间的所有块，设置预览颜色
- `endLongPressSelect` → 将预览块转为正式选中
- `cancelLongPressSelect` → 恢复预览块颜色，取消选择
