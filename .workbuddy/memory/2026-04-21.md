# 2026-04-21 工作日志

## 修复 jumpRenderDates 后 isRightNow 红框消失的问题

### 根因
`jumpRenderDates()` 跳转后会 new 出全新的 `DayRecord` 对象，新对象里所有 block 的 `isRightNow` 都是默认 `false`。
原代码没有在新建完 dayRecords 后重新调用 `updateRightNowBlock()`，导致跳回今天时红框丢失。

### 修复
在 `loaders.dart` 的 `jumpRenderDates()` 末尾，在 `notifyListeners()` 之前，补加：
```dart
userProfileLoader.updateRightNowBlock(DateTime.now());
```
与 `refreshRenderDates()` 保持完全一致。
